### Amigo Secreto

Este é um projeto desenvolvido como parte de um desafio para praticar habilidades em HTML, CSS e JavaScript. A aplicação permite que os usuários criem uma lista de nomes e, em seguida, realizem um sorteio aleatório para determinar o "amigo secreto".

#### Funcionalidades

-   **Adicionar Nomes:** Os usuários podem inserir nomes de amigos em um campo de texto. Ao clicar no botão "Adicionar", o nome é incluído em uma lista visível na tela.
-   **Validação de Entrada:** O programa valida se o campo de texto está vazio. Caso esteja, um alerta é exibido, solicitando um nome válido.
-   **Visualização da Lista:** A lista de nomes adicionados é exibida dinamicamente na página.
-   **Sorteio Aleatório:** Ao clicar no botão "Sortear Amigo", um nome é selecionado de forma aleatória e exibido na tela, simulando um sorteio de amigo secreto.

#### Tecnologias Utilizadas

-   **HTML5:** Estrutura da página.
-   **CSS3:** Estilização e layout da interface.
-   **JavaScript:** Lógica principal da aplicação, incluindo a manipulação do DOM para adicionar, visualizar e sortear os nomes.

#### Como Usar

1.  **Clone o repositório** (se estiver no GitHub) ou baixe os arquivos.
2.  Abra o arquivo `index.html` em seu navegador de preferência.
3.  Digite o nome de um amigo no campo de texto e clique em "Adicionar".
4.  Repita o processo para incluir todos os participantes.
5.  Quando a lista estiver completa, clique em "Sortear Amigo" para ver o resultado.
